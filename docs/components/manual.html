Interação com o Utilizador

A aplicação do sistema OnTrack, inicialmente, pede ao utilizador que inicie a sessão. Caso o utilizador não tenha ainda uma conta associada, este pode selecionar a opção de se registar na aplicação para ter acesso a toda a informação. está dividida em 5 secções, que se encontram no menu lateral, do lado esquerdo. Dashboard, Bus Details, Maps, Bus e User Table.

Após o início de sessão, é apresentado ao utilizador a página inicial da aplicação OnTrack. Do lado direito, o utilizador pode navegar entre cinco páginas diferentes: Dashboard (página principal), Bus Details, Maps, Bus e User Table, que serão abordadas em seguida.
