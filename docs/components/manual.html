Interação com o Utilizador

A aplicação do sistema OnTrack, inicialmente, pede ao utilizador que inicie a sessão. Caso o utilizador não tenha ainda uma conta associada, este pode selecionar a opção de se registar na aplicação para ter acesso a toda a informação. Esta está dividida em 5 secções, que se encontram no menu lateral esquerdo. Dashboard, Maps, Buses e Users Table.

Após o início de sessão, é apresentado ao utilizador a página inicial da aplicação OnTrack. Do lado direito, o utilizador pode navegar entre cinco páginas diferentes: Dashboard (página principal), Maps, Buses e Users Table, que serão abordadas em seguida.

Dashboard

Ao iniciar a sessão, o utilizador visualizará a secção Dashboard. Nesta secção, o utilizador tem acesso à velocidade média total por hora, dia, mês ou semana e a velocidade média de cada autocarro.

Maps

Nesta secção, o utilizador irá visualizar o mapa com a posição dos autocarros, em tempo real. Este mapa irá centralizar-se na cidade do Porto. No entanto, o utilizador poderá ampliar, reduzir ou arrastar o mapa de forma a visualizar melhor a área de autocarros.

Buses

Nesta secção, o utilizador poderá visualizar uma tabela com a informação atualizada dos autocarros. Para cada autocarro, é apresentado o seu ID, data e hora do registo, endereço, latitude, longitude e velocidade. Na última coluna desta tabela, quando o utilizador clicar no botão, irá ser reencaminhado para uma nova página que corresponderá ao histórico do autocarro selecionado, apresentando no mapa a sua localização e, mais uma vez, uma tabela, mas agora apenas com os registos relativos ao autocarro em causa.

Users Table

Nesta secção, o utilizador terá acesso à lista de todos os utilizadores registados na aplicação. Será apresentada uma tabela com o ID do utilizador, o respectivo username, email, city, role, rode id e last login. A cada utilizador, é permitido editar os seus dados de login ou até mesmo eliminá-los. É possível ainda adicionar um novo utilizador.
